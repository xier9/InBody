# InBody 健康紀錄

一個 iPhone 友善的 InBody 健康紀錄 PWA 原型，可記錄、儲存、比較與查看趨勢。

## 功能

- 記錄 InBody 數值
- 比較兩筆紀錄的變化
- 查看單一指標趨勢
- 匯出 JSON 備份檔
- 匯入 JSON 備份檔並依日期合併紀錄

## GitHub Pages

這是純前端靜態網站，部署到 GitHub Pages 時可直接從 `main` branch 的 root 目錄發布。

## 本機開啟

直接開啟 `index.html`，或用本機伺服器：

```bash
python3 -m http.server 4173
```

然後瀏覽 `http://localhost:4173`。
